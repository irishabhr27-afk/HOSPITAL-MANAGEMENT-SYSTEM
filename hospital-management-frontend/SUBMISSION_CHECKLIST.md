# Submission Checklist - Hospital Management System

Submission date: 28 June 2026

## Must Have

- Frontend opens without errors
- Backend server is running
- MongoDB connection is working
- Dashboard loads successfully
- Doctors page works
- Patients page works
- Appointments page works
- `README.md` is updated
- Project report content is ready
- Demo data is added

## Commands To Run Before Demo

Frontend:

```bash
npm run dev
```

Backend:

```bash
npm start
```

or:

```bash
node server.js
```

Quality check:

```bash
npm run lint
npm run build
```
