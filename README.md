# 🏥 Hospital Management System

A full-stack Hospital Management System developed as an internship project using the MERN stack. The application provides an intuitive dashboard for managing doctors, patients, and appointments with a modern, responsive user interface.

---

## 📌 Project Overview

The system simplifies hospital operations by allowing administrators to:

- Manage Doctors
- Manage Patients
- Book and Manage Appointments
- View Dashboard Analytics
- Perform Complete CRUD Operations

---

## 🚀 Features

### Frontend
- Modern Material UI Dashboard
- Responsive Design
- Dashboard Statistics
- Doctors Management
- Patients Management
- Appointment Management
- Weekly Appointment Chart
- Search & Filter
- Edit & Delete Records

### Backend
- REST API
- MongoDB Database
- Mongoose Models
- CRUD Operations
- Express.js Server
- Environment Variable Support

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/irishabhr27-afk/HOSPITAL-MANAGEMENT-SYSTEM.git
```

---

### Backend

```bash
cd hospital-management-backend

npm install

npm start
```

---

### Frontend

```bash
cd hospital-management-frontend

npm install

npm run dev
```
