## Project Description
This project is a hospital management system developed using node.js,express.js,MongoDB, and mongoose. The system helps manage hospital operations such as patient records, doctor information, and online management booking.

## Technologies Used
-Node.js
-Express.js
-MongoDB
-Mongoose
-Dotenv

## Features
-Patient Management
-Doctor Management
-online Appointment Booking
-Hospital Record Management
-REST API Backend

## How to Run

1. Install dependencies:
npm install

2. Start the server:
node server.js

3. Open browser:
https://localhost:5000
